This project presents a deep learning-based system for detecting brain tumours from MRI scan images. The model is designed to classify medical images into tumour and non-tumour categories, supporting early diagnosis and improving decision-making in clinical environments.

The solution was developed as part of a university coursework project in Robotics and Artificial Intelligence, with a focus on applying machine learning techniques to real-world healthcare problems.

Main Objectives:

- Develop a robust image classification model for brain tumour detection
- Perform data preprocessing and exploratory data analysis (EDA)
- Train and evaluate multiple machine learning models
- Optimise performance using appropriate evaluation metrics
- Demonstrate practical application of AI in healthcare

Dataset:

The dataset used in this project is publicly available and consists of labelled MRI brain scan images.

Classes:
- Tumour
- No Tumour

Data characteristics:

- Grayscale/colour MRI images
- Pre-labelled for supervised learning
- Variations in size, quality, and orientation

Results:

- The Neural Network model achieved the best overall performance
- High recall was prioritised to minimise false negatives (missing tumours)
- ROC curve analysis showed strong classification capability

Limitations: ⚠️

- Model performance depends on dataset quality
- Limited dataset size may affect generalisation
- Not suitable for real clinical deployment without further validation
